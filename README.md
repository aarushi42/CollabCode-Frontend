# CollabCode — Frontend

> A platform that connects developers worldwide to collaborate, build projects, and grow together.

🔗 **Live Demo:** [https://collabcode.in/](https://collabcode.in/)  
📦 **Backend Repo:** [github.com/aarushi42/CollabCode](https://github.com/aarushi42/CollabCode)

---

## 🚀 Tech Stack

| Technology                  | Purpose                 |
| --------------------------- | ----------------------- |
| React 19                    | UI Framework            |
| Vite 7                      | Build Tool & Dev Server |
| Tailwind CSS 4 + DaisyUI    | Styling                 |
| Redux Toolkit + React-Redux | State Management        |
| React Router DOM 7          | Client-side Routing     |
| Axios                       | HTTP Requests           |
| Socket.IO Client            | Real-time Communication |

---

---

## 🔗 Backend

This frontend connects to the [CollabCode Backend](https://github.com/aarushi42/CollabCode) — a Node.js/Express REST API with Socket.IO for real-time features.

Make sure the backend server is running before using the app locally.

## 📜 Scripts

| Command           | Description                       |
| ----------------- | --------------------------------- |
| `npm run dev`     | Start the Vite development server |

| `npm run preview` | Preview the production build      |
| `npm run lint`    | Run ESLint                        |
